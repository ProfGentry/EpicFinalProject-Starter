# Java Final Project Starter

Welcome to the Java Final Project.

You will build your project over several weeks using GitHub to track your progress. This repository includes starter code, project structure, and workflow guidance.

## Project Goal

Create a multi-class Java console application such as a game, simulator, or interactive system. Your project should grow each week as you apply new Java concepts.

## Weekly Workflow

Each week, create a new branch for that week's deliverable.

### Branch Names
- `week1-structure`
- `week2-core-actions`
- `week3-collections`
- `week4-game-logic`
- `week5-final-build`

### Weekly Process
1. Pull the latest version of your repository
2. Create the weekly branch
3. Complete the deliverable
4. Commit your work regularly
5. Push your branch to GitHub
6. Submit your GitHub link and branch name in Canvas

## Submission Expectations

Each weekly submission must include:
- Updated code in your GitHub repository
- Correct weekly branch
- Screenshot of the program running
- Short description of what you completed

## Minimum Git Expectations
- At least 3 commits each week
- Clear commit messages
- Code that compiles and runs

## Recommended Folder Structure

- `src/` → Java source files
- `docs/` → design notes and reflection
- `screenshots/` → screenshots of your progress

## Final Submission

Your final submission must include:
- Completed project in the `main` branch
- Final screenshot or demo
- Reflection file
- All required project features
